# PFE-IAQverse
IAQverse est un projet de jumeau numérique immersif en VR pour visualiser la qualité de l’air intérieur. Captation 360° avec Insta360, données IAQ en temps réel (CO₂, PM2.5…), interface web interactive et compatibilité casque VR. Backend Python (FastAPI), frontend JS (A-Frame/Three.js).
